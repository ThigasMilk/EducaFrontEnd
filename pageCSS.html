CSS: Estilizando a Web

CSS (Cascading Style Sheets) é a linguagem que controla a apresentação visual de documentos HTML. Com CSS, você pode definir cores, fontes, layouts e efeitos visuais que tornam seu site atraente e responsivo.

Vantagens do CSS

  • Separação de conteúdo e apresentação
  • Controle preciso sobre o layout
  • Design responsivo para múltiplos dispositivos
  • Animações e transições suaves

Sintaxe Básica

Um conjunto de regras CSS consiste em um seletor e um bloco de declarações:

                        
    seletor {
        propriedade: valor;
        outra-propriedade: valor;
    }

Seletores Comuns

Seletor Exemplo Descrição
Tag h1 Seleciona todos os elementos <h1>
Classe .menu Seleciona elementos com class="menu"
ID #header Seleciona o elemento com id="header"
Atributo [href] Seleciona elementos com atributo href

Box Model

O modelo de caixa é fundamental para entender layout em CSS:


    .box {
        width: 300px;
        padding: 20px;
        border: 5px solid #333;
        margin: 10px;
    }

O tamanho total do elemento será: 300px (width) + 40px (padding) + 10px (border) + 20px (margin) = 370px

Flexbox e Grid

Sistemas modernos para criar layouts complexos:


    /* Flexbox */
    .container {
        display: flex;
        justify-content: center;
        align-items: center;
    }

    /* CSS Grid */
    .grid-container {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        gap: 20px;
    }
