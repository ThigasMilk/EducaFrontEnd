JavaScript: Dando Vida à Web

JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico e interativo em páginas web. Ele permite manipular o DOM, lidar com eventos, fazer requisições assíncronas e muito mais.

Por que JavaScript é essencial?

  • Última linguagem executada diretamente no navegador
  • Permite interações complexas em tempo real
  • Base para frameworks modernos como React e Vue
  • Possível usar no back-end com Node.js

Conceitos Básicos

                        
    // Variáveis
    let mensagem = "Olá, Mundo!";
    const PI = 3.14;

    // Funções
    function soma(a, b) {
        return a + b;
    }

    // Estruturas de controle
    if (idade >= 18) {
        console.log("Maior de idade");
    } else {
        console.log("Menor de idade");
    }

    // Loops
    for (let i = 0; i < 5; i++) {
        console.log(i);
    }

Manipulação do DOM

JavaScript pode alterar elementos HTML dinamicamente:

                        
    // Selecionar elemento
    const botao = document.getElementById('meuBotao');

    // Manipular conteúdo
    botao.textContent = 'Novo Texto';

    // Adicionar evento
    botao.addEventListener('click', function() {
        alert('Botão clicado!');
    });

    // Criar novo elemento
    const novoParagrafo = document.createElement('p');
    novoParagrafo.textContent = 'Texto dinâmico';
    document.body.appendChild(novoParagrafo);
