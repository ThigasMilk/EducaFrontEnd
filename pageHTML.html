HTML: A Estrutura da Web

HTML (HyperText Markup Language) é a linguagem fundamental para criação de páginas web. Ele fornece a estrutura básica de um site, organizando o conteúdo em elementos hierárquicos que os navegadores podem interpretar e renderizar.

Por que aprender HTML?

  • É a base de todo desenvolvimento web
  • Facilitação do SEO com tags semânticas
  • Integração simples com CSS e JavaScript
  • Necessário para qualquer framework front-end

Estrutura Básica

Todo documento HTML começa com uma estrutura básica:

                        

    <!DOCTYPE html>
    <html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título da Página</title>
    </head>
    <body>
        <!-- Conteúdo da página aqui -->
    </body>
    </html>

Elementos Semânticos

HTML5 introduziu elementos semânticos que melhoram a acessibilidade e SEO:

                        
                            
    <header>Cabeçalho da página</header>
    <nav>Navegação principal</nav>
    <main>
        <article>
            <h1>Título do Artigo</h1>
            <p>Conteúdo principal</p>
        </article>
    </main>
    <aside>Conteúdo relacionado</aside>
    <footer>Rodapé</footer>

Principais Tags

  • <h1>-<h6>: Títulos e subtítulos
  • <p>: Parágrafos de texto
  • <a>: Links para outras páginas
  • <img>: Inserção de imagens
  • <ul>/<ol>: Listas não ordenadas/ordenadas
  • <div>: Contêiner genérico
